## 📌 Project Overview
Part of the **IBM Data Science Capstone** — scrapes historical Falcon 9 and Falcon Heavy launch records from Wikipedia using BeautifulSoup. Extracts launch tables, cleans HTML, and structures the data into a Pandas DataFrame as an alternative data collection method to the SpaceX API.

**Domain:** Web Scraping — Space Tech  
**Source:** Wikipedia — List of Falcon 9 and Falcon Heavy launches  
**Goal:** Falcon 9 first stage landing prediction  

## 🎓 Skills Demonstrated
Wikipedia web scraping · BeautifulSoup table extraction · HTML table parsing · Pandas DataFrames · IBM DS Capstone
